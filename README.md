# Tournament Match Schedule

A premium match schedule application for tracking cricket tournaments. Featuring a modern, dynamic UI with glassmorphism effects and smooth micro-animations.

## 🚀 Features
- **Modern UI**: Sleek design with a dark mode aesthetic and vibrant gradients.
- **Dynamic Schedule**: Clear visibility of upcoming matches and team standings.
- **Responsive Design**: Works perfectly across all device sizes.
- **Premium Aesthetics**: High-quality team logos and smooth hover interactions.

## 🛠️ Tech Stack
- **React**: Core UI framework
- **Tailwind CSS**: Modern styling and layout
- **Vite**: Ultra-fast build tool and dev server
- **JavaScript (ES6+)**: Logic and interactivity

## 📦 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yagnesh-rajpara/vrajnidhi-box.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
